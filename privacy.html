Sound Pilot - Privacy Policy

Last Updated: November 9, 2025

Single Purpose Description

Sound Pilot is a browser extension that provides enhanced audio volume control for media content on supported websites. The extension allows users to amplify audio beyond browser limits (up to 20x), control playback, customize themes, and manage volume across multiple tabs - all through a simple popup interface.

Chrome Web Store Privacy Disclosures

Single Purpose

Enhanced audio volume control and playback management for browser media content.

Sound Pilot serves one clear purpose: to give users complete control over audio volume and playback in their browser tabs. The extension amplifies audio beyond standard browser limits, provides preset and custom volume multipliers, and offers convenient playback controls for supported media platforms.

Permissions Justification

Required Permissions:

1. activeTab

  • Purpose: Access the currently active tab to detect and control media elements
  • Usage: Allows the extension to inject volume control scripts into the active tab when the user clicks the extension icon
  • Data Access: Only accesses the tab when user explicitly opens the popup

2. tabs

  • Purpose: List and identify tabs with media content for multi-tab control
  • Usage: Enables the dropdown selector that shows all tabs with active media
  • Data Access: Only reads tab titles and URLs to identify media platforms (YouTube, Spotify, etc.)

3. scripting

  • Purpose: Inject content scripts to control audio elements on web pages
  • Usage: Required to manipulate Web Audio API and HTML5 media elements for volume control
  • Data Access: Only injects scripts into supported media platforms

4. storage

  • Purpose: Save user preferences locally
  • Usage: Stores volume settings, theme preferences, and last-used multiplier
  • Data Access: All data stored locally in browser; never transmitted

Host Permissions:

The extension requests access to specific media platforms:

  • https://www.youtube.com/*
  • https://open.spotify.com/*
  • https://music.youtube.com/*
  • https://soundcloud.com/*
  • https://www.netflix.com/*
  • https://www.twitch.tv/*
  • https://vimeo.com/*

Purpose: These permissions allow the extension to detect and control audio/video elements on these specific platforms. The extension only activates on these domains and does not access any other websites.

Data Collection and Usage

What We DO NOT Collect:

  • ❌ Personal information (name, email, address)
  • ❌ Browsing history
  • ❌ Website content or data
  • ❌ Cookies or tracking data
  • ❌ Analytics or usage statistics
  • ❌ Any data transmitted to external servers

What We Store Locally:

  • ✅ Volume multiplier preferences (e.g., 2x, 5x)
  • ✅ Base volume level (0-100%)
  • ✅ Selected theme preference
  • ✅ Last used settings per tab

Storage Location: All data is stored locally in your browser using Chrome's chrome.storage.local API. This data never leaves your device.

Data Transmission

Sound Pilot does NOT transmit any data.

  • No network requests are made to external servers
  • No analytics or tracking services are used
  • No user data is collected, shared, or sold
  • All functionality operates entirely within your browser

Third-Party Services

Sound Pilot does not use any third-party services, analytics, or tracking tools.

User Data Privacy

Data Access:

  • The extension only accesses media elements (audio/video tags) on supported websites
  • No access to passwords, form data, or personal information
  • No access to page content beyond media controls

Data Retention:

  • Settings are stored indefinitely in local browser storage
  • Users can clear all data by removing the extension
  • No data is retained after uninstallation

Data Sharing:

  • No data is shared with third parties
  • No data is transmitted to developers or any external service
  • All processing happens locally in your browser

Security

  • Uses Chrome's secure message passing API for communication between extension components
  • Follows Chrome Extension Manifest V3 security best practices
  • Minimal permission requests (only what's necessary for functionality)
  • No external dependencies or libraries that could compromise security

Children's Privacy

Sound Pilot does not knowingly collect any information from users of any age. The extension does not collect, store, or transmit personal data.
